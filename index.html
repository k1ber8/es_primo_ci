Bienvenido al Proyecto es_primo

Validación de números primos con pruebas automatizadas

Este cambio prueba la integración continua con GitHub Actions.

Descripción

Este proyecto tiene como objetivo verificar si un número es primo utilizando Python. Además, se implementan pruebas automatizadas con pytest y un flujo de integración continua (CI) mediante GitHub Actions para asegurar el correcto funcionamiento del código.

Las pruebas validan diferentes casos como números primos, números no primos y manejo de errores cuando se ingresan valores inválidos.
